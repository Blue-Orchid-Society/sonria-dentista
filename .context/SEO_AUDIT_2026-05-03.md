## What's actually winning on-page (the patterns)

### 1. Title tag formula
The top 5 results across all four cities all use this template:
> `[Adjective/Brand] Dentist in [City], TX [| or –] [Brand or Benefit]`

Examples:
- "Gentle Care Dentist in Paris TX for Your Family's Smiles"
- "Family Dentist in Paris, TX - Oak Creek Family Dental"
- "Commerce Family Dental – Dental Professionals in Commerce, TX"
- "Dentist in Grand Prairie, TX | Great Southwest Family Dental"

**Length**: 53-61 characters. Always includes city + TX in the keyword position.

**Our current title**: `"Sonria Dentista | Bilingual family dental care in North Texas"` — does NOT include any specific city. The home page can therefore rank for none of the four target cities, only the broad "north texas" phrase which has very thin volume. This is the **biggest SEO bug in our current rebuild.**

### 2. H1 formula
> `Your Trusted Dentist in [City], TX[: optional benefit subtitle]`

Examples (all #1 ranking):
- "Your Trusted Dentist in Commerce, TX"
- "Your Trusted Dentist in Paris, TX: Personalized Care for Every Smile"

**Our current H1**: home hero title is brand statement, NOT a keyword-bearing H1. Location pages do hit `[Service] in [City], TX` correctly — this part is fine.

### 3. Section grammar that ranks
This is the consistent post-hero structure across the top-ranking pages:

```
1. Hero (H1 + photo + 2 CTAs)
2. Trust band (3 short props: hours/insurance/star rating)  ← we don't have this
3. Welcome paragraph (brand + city + people-first language)  ← we don't have this
4. Services grid (5-9 services as H3 tiles)  ← we have this
5. Why Choose Us (bullet list of differentiators)  ← we have this on detail pages, not home
6. Meet Doctor / Team (photos + bios)  ← we have this
7. Testimonials with patient names  ← WE DON'T HAVE THIS
8. Office info (NAP block: address + hours + phone)  ← partial, in footer only
9. Blog / News / Did-You-Know  ← WE DON'T HAVE THIS
10. Booking CTA band  ← we have this
```

### 4. Word count target
- Top Paris result: **894 words** on home page
- Top Commerce result: **650 words**
- Top Grand Prairie result: **1051 words**

**Our current home page**: ~250 words across all sections. **Too thin** — Google's local-business heuristics expect 600+ words on a service-business home page.

### 5. Schema markup
**Every** top-ranking competitor has `LocalBusiness` + `Dentist` schema markup with NAP, opening hours, and aggregate rating. **Our site has none.** This is the single highest-leverage technical fix.

## Recommended diff (Sonria — priority order)

### P0 — SEO/structural (does not require redesign)
1. Per-locale, per-page-template metadata: home title becomes `"Bilingual Dentist in Arlington TX | Sonria Dentista"` (or rotate by primary location). Add `og:title` + `twitter:title` parity.
2. Add `LocalBusiness` JSON-LD on every page, `Dentist` + `MedicalBusiness` types, with NAP, hours, `aggregateRating` from GBP, `areaServed` for the four cities.
3. Add per-location JSON-LD on `/locations/[slug]` pages with that location's specific NAP.
4. Add `BreadcrumbList` JSON-LD.
5. Verify URL paths from old site don't break — pull current sitemap from `sonriadentista.com`, redirect any high-equity URLs.

### P1 — Content depth (visible changes)
6. Add **trust band** right under hero: 3 short value props ("Same-day visits" / "Spanish + English" / "All major insurance"), pulled from current data, rendered as 3-column strip with thin top/bottom borders.
7. Add **welcome paragraph** below trust band (200-word intro mentioning all 4 cities + the bilingual hook + Dr. Namineni).
8. Add **patient testimonials** section — real Google reviews from the GBP, 3-card carousel with names + 5-star + city tag.
9. Add **NAP block** with all 4 location addresses + hours + phone, matching GBP exactly.
10. Add **"Did You Know" / FAQ teasers** at home — 3 cards linking to `/faq#anchor` (capture People-Also-Ask intent).

### P2 — Visual differentiation from Chapala
11. Change Sonria's display font from Playfair Display to something dental-warm (DM Serif Display, Fraunces, or a softer humanist) — Chapala keeps Playfair.
12. Change Sonria's accent from terracotta to a warmer coral or muted teal — Chapala keeps terracotta.
13. Break the eyebrow→H2→subhead repetition: alternate section types (full-bleed image, asymmetric 2-col with overlap, dark band, light band).
14. Vary section padding (some `py-16`, some `py-24`, some `py-32`) instead of uniform.

### P3 — Content engine
15. Build `/articles` route reading from `data/articles/*.json`, seed with 5-10 SEO articles answering People-Also-Ask topics: "How much do dental implants cost in Texas?", "Dentist that accepts Medicaid in Arlington TX", "Same-day crowns explained", etc.
16. Wire SEO agent to push article JSON via PR.
